# Software Development Tools and Environment Lab

소프트웨어 개발 과정에서 사용되는  
**개발 도구, 빌드 환경, 자동화 흐름을 실습을 통해 학습한 기록**

본 저장소는 *소프트웨어개발도구및환경실습* 과목에서  
**6주차부터 13주차까지 수행한 실습 및 과제를 정리한 학습 레포지토리**입니다.

각 주차별로 예비보고서, 결과보고서, 실습 코드를 통해  
개발 도구와 환경이 실제 개발 과정에서 어떻게 활용되는지를 학습했습니다.

## 📌 과목 개요

- **과목명**: 소프트웨어개발도구및환경실습
- **학습 목표**
  - 소프트웨어 개발 도구의 역할 이해
  - 빌드 및 실행 환경 구성 경험
  - 자동화 도구를 통한 개발 효율성 향상

---

## 🧩 학습 범위

- **6주차 ~ 13주차**
- 각 주차별로:
  - 예비보고서
  - 결과보고서
  - 실습 코드 포함
- 개발 도구 사용 경험과 환경 설정 과정을 중심으로 정리

---

## 🛠 기술 스택
Language : C / C++
Tools : gcc, Makefile
Environment : Linux

---

## 🧠 주차별 학습 내용

### 🔹 6주차
- 개발 환경 설정
- 기본 컴파일 및 실행 과정 이해
- 개발 도구 사용 흐름 파악

### 🔹 7주차
- 컴파일 과정 단계별 분석
- 컴파일 옵션에 따른 결과 비교
- 오류 및 경고 메시지 해석

### 🔹 8주차
- Makefile 작성 및 활용
- 빌드 자동화 개념 학습
- 의존성 관리의 중요성 이해

### 🔹 9주차
- 코드 변경에 따른 빌드 흐름 분석
- 반복 작업 자동화 실습
- 개발 생산성 관점에서 도구 활용

### 🔹 10주차
- 실습 내용 종합 정리
- 빌드 및 실행 과정 전체 흐름 이해
- 개발 도구 사용 효과 분석

### 🔹 11주차
- 실습 코드 확장 및 응용
- 빌드 구조 이해 심화
- 개발 환경 설정에 따른 차이 분석

### 🔹 12주차
- 복합 빌드 환경 실습
- 여러 파일을 포함한 빌드 과정 이해
- 실습 결과 분석 및 정리

### 🔹 13주차
- 전체 실습 내용 종합
- 개발 도구 및 환경의 역할 정리
- 실제 개발 과정에서의 활용 관점 정리

---

## 🔄 학습 방식

1. 예비보고서를 통한 개념 및 실습 목표 정리  
2. 실습 코드를 직접 작성하며 도구 사용  
3. 결과보고서를 통해 실습 결과 분석  
4. 개발 환경과 개발 효율의 관계를 중심으로 학습
